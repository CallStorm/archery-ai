

## Archery AI 简介

Archery AI 是一个在开源 Archery 基础上**深度融合 AI 能力**的 SQL 审查与查询平台。

---

## 🚀 改造目的：AI 驱动的研发效能革新

本次改造的核心目标在于：**在不改变现有流程与架构的前提下，通过引入尖端 AI 特性，实现 SQL 审查的智能化增强，最终达到提高研发效能、降低使用门槛的目的。**

我们致力于将 Archery AI 打造成一个更智能、更易用、更高效的 SQL 治理平台，同时这也是我们在 **“AI 融合传统应用”** 这一前沿领域的一次积极探索。

### 1. 🌟 **增强易用性与体验**
* 在开源基础之上**新增 AI 特性**，让 **SQL 审查更易使用**，降低用户理解和操作的复杂度。
* 通过智能化的辅助，**降低使用成本**，让非资深 DBA 也能轻松完成高质量的 SQL 审查。

### 2. ⚡️ **提升研发效能**
* 以**研发效能为核心指标**，通过自动化、智能化的手段，**缩短评审与迭代周期**。
* 减少开发人员和 DBA 之间因 SQL 规范问题而产生的**重复沟通与手工评审时间**，加速应用交付。

### 3. 🧠 **探索 AI 融合路径**
* 探索 AI 与传统应用的融合路径，**在不改变现有架构与流程的前提下进行智能增强**，为现有系统注入 AI 动力。

---

## 🎯 拟引入的 AI 特性（规划）

| 核心能力 | 描述 | 预期效果 |
| :--- | :--- | :--- |
| **智能风险识别** | **智能 SQL 语义理解**与风险识别，输出清晰、**可解释**的审查结论。 | **更好质量**：风险识别更全面、建议更可执行，提升稳定性与合规性。 |
| **自然语言交互** | **自然语言到 SQL 的意图解析**与建议生成，极大提升查询与审查的**可用性**。 | **更低门槛**：无需深入理解复杂规则即可完成高质量审查。 |
| **审查建议与修复** | 提供**审查建议与自动修复提示**，为常见问题提供一键修复或可执行替代方案。 | **更高效率**：加速问题解决，减少人工干预。 |
| **持续策略优化** | 从历史变更与事件中学习**隐性规则**，持续优化审查策略与风控规则。 | **更佳适应性**：审查策略随业务发展持续优化。 |
